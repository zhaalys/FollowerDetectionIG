##  Fitur Unggulan

- ✅ **Cek Unfollowers**: Lihat daftar orang yang kamu follow tapi gak follow balik.
- ✅ **Cek Fans**: Temukan orang yang nge-fans (follow kamu tapi kamu gak follow).
- ✅ **Cek Mutual**: Lihat siapa saja teman sejati (saling follow).
- ✅ **Login Otomatis**: Cukup login sekali, besok-besok tinggal buka tanpa ketik password lagi! (Sistem _Session Cookies_).
- ✅ **Amannn**: Password tersimpan di laptop sendiri, gak dikirim ke server manapun.
- ✅ **Bypass IP Block (Beta)**: Dilengkapi fitur penyamaran lokasi (Indonesia) agar lebih aman dari blokir Instagram.

##  Cara Install & Pakai (Gampang Banget!)

Pastikan kamu sudah install [Python](https://www.python.org/downloads/) ya!

### 1. Clone Project Ini

Download atau clone repository ini:

```bash
git clone https://github.com/zhaalys/FollowerDetectionIG.git
cd FollowerDetectionIG
```

### 2. Install Library

Jalanin perintah ini di terminal/cmd:

```bash
pip install -r requirements.txt
```

### 3. Jalankan Aplikasi

Ketik ini dan tunggu browser terbuka otomatis:

```bash
streamlit run app.py
```

##  Cara Menggunakan

1.  **Login**: Masukkan Username & Password Instagram kamu di menu sebelah kiri.
    - _Tips: Gunakan akun kedua (second account) buat login biar akun utama makin aman!_
2.  **Target**: Mau cek akun siapa?
    - Tulis username akun utama kamu di kolom "Target Username".
3.  **Analisa**: Klik tombol **"Analisa Sekarang"**.
4.  **Tunggu**: Biarkan bot bekerja mengambil data (lamanya tergantung koneksi internet).
5.  **Hasil**: Lihat tab "Tidak Follback", "Fans", dan "Mutual".

##  Peringatan (Disclaimer)

- **Gunakan dengan Bijak**: Jangan spam cek 100x sehari nanti dikira bot sama Instagram.
- **IP Block**: Kalau tiba-tiba gagal login ("Feedback Required" / "Blacklist"), itu artinya IP WiFi kamu diblokir sementara. Ganti ke **Hotspot HP (Data Seluler)** sebentar, lalu coba lagi.
- **Resiko**: Penggunaan bot pihak ketiga selalu memiliki resiko. Penulis tidak bertanggung jawab atas akun yang terkena limit/suspensi.
